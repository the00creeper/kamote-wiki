# 🛠️ Technical Information

<figure><img src="../../.gitbook/assets/image (32).png" alt=""><figcaption></figcaption></figure>

This section of the wiki provides technical information about the server's internal configuration. These settings affect many aspects of gameplay, including:

* Mob spawning behavior
* Entity activation and tracking
* Redstone mechanics
* Hopper and item systems
* Despawn distances
* Crop growth rates
* Portal mechanics

The goal of publishing these settings is **transparency** and **technical accessibility**. Many players design farms and contraptions that rely on precise game mechanics.&#x20;

By making relevant configuration values public, players can better understand how the server behaves and design farms that perform reliably.
