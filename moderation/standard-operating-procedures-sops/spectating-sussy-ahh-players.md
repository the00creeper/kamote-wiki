# 👀 Spectating Sussy Ahh Players

A player is sussy ahh if they:

* Are repeatedly flagged by **Vulcan**
* Send suspicious `/msg` whispers
* Threaten others in chat
* Exhibit behavior suggesting alternate accounts

### 📝 **Procedure**

1. Enter Moderation Mode: `/mon <reason>`
2. Teleport to the player: `/tp <player>`
3. Observe and document their actions through screenshots or recordings.
4. Return to your original location: `/back`
5. Exit Moderation Mode: `/moff`

{% hint style="warning" %}
Always return to your original position after teleporting.
{% endhint %}

### 💡 **Useful Commands**

* `/invsee <player>` — View a player's inventory
* `/endersee <player>` — View a player's Ender Chest
* `/gms` — Switch to Survival Mode
* `/gmsp` — Switch to Spectator Mode

## **🔎 Inspecting with CoreProtect**

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure></div>

* Use `/co i` to inspect block history.

{% hint style="warning" %}
This command does **not** work in **Spectator Mode**.
{% endhint %}

* Switch to Survival Mode using `/gms` to inspect blocks.
* Toggle the inspector off by running `/co i` again before exiting Moderation Mode.

***

## 📦 **Opening Containers**

* Containers can be accessed in both Survival and Spectator Modes.

{% hint style="danger" %}
**Do not open containers in Survival Mode**, as nearby players may hear the interaction.
{% endhint %}

* Always access containers while in Spectator Mode for discretion.
