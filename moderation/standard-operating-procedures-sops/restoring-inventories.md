# 📦 Restoring Inventories

Inventory restoration is permitted **only** when losses result from server-related issues.

### ✅ **Valid Cases**

* Server glitches or restarts
* Technical faults **beyond** the player's control

### 📝 **Procedure**

1. Ask the player how and when the loss occurred.
2. Verify the claim.
3.  Restore the inventory: `/irp restore <player>`<br>

    <div data-with-frame="true"><figure><img src="../../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure></div>


4. Review relevant records within the GUI:
   * Deaths
   * Joins
   * Quits
   * World changes (e.g., Overworld → Nether)
   * Force backups

{% hint style="success" %}
Select the record that best matches the player’s report.
{% endhint %}

### 🚫 **Invalid Cases**

Do **not** restore inventories for:

* Mob deaths
* AFK deaths
* Void falls
* Warden encounters
* Player negligence

{% hint style="info" %}
Players are responsible for their actions. Maintain fairness at all times.
{% endhint %}
