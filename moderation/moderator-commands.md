# 📋 Moderator Commands

### Argument Types

* `<argument>` - required
* `[argument]` - optional

| Command                                 | Description                                                                            |
| --------------------------------------- | -------------------------------------------------------------------------------------- |
| `/alts <player>`                        | Show other accounts linked to a player's IP address                                    |
| `/back`                                 | Returns you back to your previous location before teleporting                          |
| `/ban <player> [reason]`                | Bans a player                                                                          |
| `/bc <message>`                         | Broadcasts a message to all players                                                    |
| `/co i`                                 | Enters inspector mode (toggleable)                                                     |
| `/co lookup <arguments>`                | Looks up player and world changes from the given [arguments](coreprotect-arguments.md) |
| `/co near`                              | Looks up player and world changes around you                                           |
| `/discord unlink <player>`              | Unlinks a player's Discord account from Minecraft                                      |
| `/donatoradd <player>`                  | Adds the DONATOR role to a player                                                      |
| `/donatorremove <player>`               | Removes the DONATOR role to a player                                                   |
| `/endersee <player>`                    | Sees the ender chest of a player                                                       |
| `/gms`                                  | Enters survival mode                                                                   |
| `/gmsp`                                 | Enters spectator mode                                                                  |
| `/invsee <player>`                      | Sees the live inventory of a player                                                    |
| `/ignoreclaims`                         | Ignores claims, allowing you to do anything inside them (toggleable)                   |
| `/irp restore <player>`                 | Restores a player's inventory to a chosen time                                         |
| `/kick <player> [reason]`               | Kicks a player                                                                         |
| `/mon <reason>`                         | Enters Moderation Mode                                                                 |
| `/moff`                                 | Exits Moderation Mode                                                                  |
| `/mute <player> [duration] [reason]`    | Mutes a player                                                                         |
| `/rankset <player> <rank>`              | Sets the rank of a player                                                              |
| `/saferestart`                          | Restarts the server given a 30-second preparation period                               |
| `/stc`                                  | Enters Staff Chat mode (toggleable)                                                    |
| `/tempban <player> <duration> [reason]` | Temporarily bans a player                                                              |
| `/tp <player>`                          | Teleports instantly to a player                                                        |
| `/unban <player>`                       | Unbans a banned player                                                                 |
| `/unregister <player>`                  | Unregisters a player                                                                   |
