## 💲 Standard Retail Prices (SRP) Implemented

<figure><img src="../../.gitbook/assets/image (1) (1).png" alt=""><figcaption></figcaption></figure>

Implemented na ang **SRP** sa mga items na napag-usapan natin nitong nakaraang staff-player meeting.

* Ang SRP ay applicable lang sa mga **rare or difficult-to-obtain** na items.
* **Malaya pa rin** ang galawan ng presyo sa mga items na **wala sa listahan** ng SRP.

Mahalaga ito para maiwasan ang unnecessary overpricing sa mga rare items, para masigurong accessible pa rin ito lalo na sa mga new players.

## :gear: **Technical Server Configurations**

<figure><img src="../../.gitbook/assets/image (2) (1).png" alt=""><figcaption></figcaption></figure>

Mayroon nang bagong wiki page na dedicated sa lahat ng **technical server configuration files**.

Mahalaga ito para sa mga **farm builders, redstone engineers, and technical players** na kailangan ng accurate server mechanics to create **efficient designs**.

Ang mga available na configuration references ay:

* `server.properties`
* `spigot.yml`
* `bukkit.yml`
* `paper-global.yml`
* `paper-world-defaults.yml`

Mahalaga ang mga config na ito para malaman ang:

* Mob spawning behavior
* Entity activation and tracking ranges
* Despawn distances
* Iba pang technical mechanics

## 💬 **Discord Linking Lookup**

May bagong command na nadagdag: `/discord linked <player OR username>` Ang ginagawa nito ay:

* Malaman ang **Discord account** ng isang **Minecraft player**
* Pwede ring **Discord username** ang i-input para malaman ang **Minecraft account** niya

Mahalaga ito para mas mapadali ang **paghahanap ng contact** sa mga players, both inside and outside the game, nang hindi na kailangan pa ng pagtatanong.
