# ✅ Patch 3.1

* Pwede na magamit ang `/spawn` to teleport you back to spawn. This has a 40 minute cooldown for all ranks.
* Teleport delay has been adjusted from 3 seconds to 5 seconds.
* Hindi ka na tatablan ng kahit anong damage while in the pathway region sa spawn papunta sa admin base. This is a great refuge for players na nagsisimula pa lang!
